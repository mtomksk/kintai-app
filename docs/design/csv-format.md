# CSVエクスポート仕様

> 勤務実績画面（A-02）の [CSVエクスポート] で出力するファイルの仕様。

---

## 出力ファイル

| 項目 | 仕様 |
|------|------|
| ファイル形式 | CSV（カンマ区切り） |
| 文字コード | UTF-8（BOM付き）※ Excelで日本語が文字化けしないため |
| ファイル名 | `勤務実績_YYYY年MM月.csv`（例: `勤務実績_2026年07月.csv`） |
| 改行コード | CRLF |

---

## CSV①: 給与計算用

### 用途
- 月次の給与計算
- 遅刻・早退のチェック
- 労働時間の法令遵守確認

### カラム定義

| # | カラム名 | 型 | 算出元 | 説明 |
|---|---------|-----|------|------|
| 1 | 作業員名 | string | Worker.name | — |
| 2 | 作業員ID | string | Worker.userId | 一意識別 |
| 3 | 日付 | date | DailyRecord.date | YYYY-MM-DD形式 |
| 4 | 曜日 | string | 日付から算出 | 月〜日 |
| 5 | 出勤時刻 | time | TimeStamp(ClockIn) | HH:MM形式 |
| 6 | 退勤時刻 | time | TimeStamp(ClockOut) | HH:MM形式 |
| 7 | 総勤務h | number | 退勤 - 出勤 | 小数（例: 8.5） |
| 8 | 休憩h | number | ⏸️ 保留 | 小数（例: 1.0） |
| 9 | 実労働h | number | 総勤務 - 休憩 | 小数（例: 7.5） |
| 10 | 現場数 | number | SiteStartの回数 | 整数 |
| 11 | 移動h | number | 現場間の空き時間合計 | 小数 |

### 出力例

```csv
作業員名,作業員ID,日付,曜日,出勤時刻,退勤時刻,総勤務h,休憩h,実労働h,現場数,移動h
田中太郎,worker001,2026-07-01,火,09:00,17:00,8.0,1.0,7.0,3,0.5
田中太郎,worker001,2026-07-02,水,08:30,16:30,8.0,1.0,7.0,2,0.3
マリク,worker003,2026-07-01,火,08:30,17:30,9.0,1.0,8.0,2,0.5
```

### Excelでの使い方

1. CSVをExcelで開く
2. L列に「時給」を手入力（例: ¥1,200）
3. M列に `=I2*L2` で日次給与を算出
4. 月次合計は `=SUMIF(A:A,"田中太郎",M:M)` で算出

## CSV②: 人事評価用

### 用途
- 作業員の評価材料
- 現場ごとの作業品質・スピードの把握

### カラム定義

| # | カラム名 | 型 | 算出元 | 説明 |
|---|---------|-----|------|------|
| 1 | 作業員名 | string | Worker.name | — |
| 2 | 作業員ID | string | Worker.userId | 一意識別 |
| 3 | 日付 | date | DailyRecord.date | YYYY-MM-DD形式 |
| 4 | 現場名 | string | Site.name | — |
| 5 | 清掃難易度 | string | Site.difficulty | ★☆☆ / ★★☆ / ★★★ |
| 6 | 作業h | number | SiteEnd - SiteStart | 小数 |
| 7 | 担当箇所 | string | WorkReport.areas | セミコロン区切り |
| 8 | 清掃品質 | string | WorkReport.quality | A / B / C |
| 9 | 備考（翻訳） | string | WorkReport.notes.translated | 日本語訳 |
| 10 | 備考（原文） | string | WorkReport.notes.original | 入力言語のまま |

### 出力例

```csv
作業員名,作業員ID,日付,現場名,清掃難易度,作業h,担当箇所,清掃品質,備考（翻訳）,備考（原文）
田中太郎,worker001,2026-07-01,新宿マンションA,★★★,2.5,台所;浴室,A,特になし,特になし
田中太郎,worker001,2026-07-01,渋谷ハイツB,★★☆,3.0,リビング;トイレ,B,汚れがひどかった,汚れがひどかった
マリク,worker003,2026-07-01,新宿マンションA,★★★,2.0,玄関;廊下,A,浴室が大変でした,බාත්රූම් එක අමාරුයි
```

---

## 設計上の注意点

| 項目 | 方針 |
|------|------|
| 時間の形式 | 小数（8.5）を使用。HH:MM形式はExcelでの計算が面倒なため |
| 出勤・退勤時刻 | HH:MM形式のまま。給与計算には直接使わないが参照用に残す |
| 欠勤日 | 出力しない（出勤データがある日のみ行を出力） |
| 休憩時間 | ⏸️ 保留事項の解決後に仕様を確定する |
| 並び順 | 作業員名 → 日付 の昇順 |
