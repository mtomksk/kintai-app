# 画面遷移図（Mermaid版）

> `screens.md` の画面一覧・遷移テーブルに基づき、Mermaid形式で図化したもの。
> GitHubで図として表示されます。

---

## 1. 全体概要（ログイン〜ロール分岐）

```mermaid
flowchart TD
    S01["S-01: ログイン画面"]

    S01 -->|"role = worker"| W01["W-01: 打刻画面（ホーム）"]
    S01 -->|"role = admin"| A01["A-01: 管理ダッシュボード"]

    W01 -.->|"ログアウト"| S01
    A01 -.->|"ログアウト"| S01
    A01 -->|"自分の打刻タブ"| W01

    style S01 fill:#6366f1,stroke:#4f46e5,color:#fff
    style W01 fill:#dbeafe,stroke:#3b82f6
    style A01 fill:#ffedd5,stroke:#f97316
```

---

## 2. 作業員フロー（🔵 Worker）

```mermaid
flowchart TD
    W01["W-01: 打刻画面\n（ホーム）"]
    W02["W-02: 案件選択画面"]
    W03["W-03: 作業報告画面"]
    W04["W-04: 勤務履歴画面"]
    W05["W-05: 設定画面"]

    W01 -->|"現場開始"| W02
    W02 -->|"案件を選択して作業開始"| W01
    W02 -->|"現場が一覧にない → スキップして現場開始"| W01

    W01 -->|"現場終了 → 報告する"| W03
    W03 -->|"報告を送信"| W01

    W01 -->|"未提出バッジ → 報告する"| W03

    W01 -->|"履歴"| W04
    W04 -->|"戻る"| W01

    W01 -->|"設定"| W05
    W05 -->|"戻る"| W01

    style W01 fill:#dbeafe,stroke:#3b82f6,stroke-width:3px
    style W02 fill:#dbeafe,stroke:#3b82f6
    style W03 fill:#dbeafe,stroke:#3b82f6
    style W04 fill:#dbeafe,stroke:#3b82f6
    style W05 fill:#dbeafe,stroke:#3b82f6
```

**ポイント**:
- W-01（打刻画面）がハブ。すべての画面はW-01を経由する
- 現場終了後、作業報告を促す → W-03 へ遷移
- 未提出の報告がある場合、バッジからもW-03 へ遷移可能

## 3. 管理者フロー（🟠 Admin）

```mermaid
flowchart TD
    A01["A-01: 管理ダッシュボード\n（勤務時間確定）"]
    A02["A-02: 勤務実績画面\n（月単位）"]
    A03["A-03: 作業員詳細画面"]
    A04["A-04: 現場管理画面\n（物件/案件）"]
    A05["A-05: 割り振り画面"]
    A06["A-06: アカウント管理画面"]
    A07["A-07: 設定画面"]

    A01 -->|"勤務実績"| A02
    A01 -->|"作業員をタップ"| A03
    A01 -->|"現場管理"| A04
    A01 -->|"割り振り"| A05
    A01 -->|"アカウント"| A06
    A01 -->|"設定"| A07

    A02 -->|"作業員をタップ"| A03
    A02 -->|"戻る"| A01
    A03 -->|"戻る"| A01
    A04 -->|"戻る"| A01
    A05 -->|"戻る"| A01
    A06 -->|"戻る"| A01
    A07 -->|"戻る"| A01

    style A01 fill:#ffedd5,stroke:#f97316,stroke-width:3px
    style A02 fill:#ffedd5,stroke:#f97316
    style A03 fill:#ffedd5,stroke:#f97316
    style A04 fill:#ffedd5,stroke:#f97316
    style A05 fill:#ffedd5,stroke:#f97316
    style A06 fill:#ffedd5,stroke:#f97316
    style A07 fill:#ffedd5,stroke:#f97316
```

**ポイント**:
- A-01（ダッシュボード）がハブ。全画面はA-01から遷移
- A-02（勤務実績）からもA-03（作業員詳細）へ遷移可能
- A-01は [管理] / [自分の打刻] のタブ切替あり（打刻タブは W-01 相当）

---

## 凡例

| 記号 | 意味 |
|------|------|
| 実線矢印 `→` | 通常の画面遷移（ボタン操作等） |
| 点線矢印 `⇢` | ログアウト（確認ダイアログあり） |
| 🔵 青ノード | 作業員向け画面 |
| 🟠 橙ノード | 管理者向け画面 |
| 🟣 紫ノード | 共通画面（ログイン） |
| **太枠** | ハブ画面（各ロールの起点） |

---

> 詳細な遷移条件（ボタンごとの遷移先・条件）は [screens.md](./screens.md) の「画面遷移図（詳細：ボタン → 遷移先）」セクションを参照。
