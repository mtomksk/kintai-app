## 画面遷移図（詳細：ボタン → 遷移先）

### S-01: ログイン画面

| 操作 / UI要素 | 遷移先 | 条件 |
|---|---|---|
| [ログイン] ボタン押下 | **W-01** 打刻画面 | role = worker の場合 |
| [ログイン] ボタン押下 | **A-01** 管理ダッシュボード | role = admin の場合 |
| [ログイン] ボタン押下 | **S-01**（エラー表示） | 認証失敗の場合 |

---

### W-01: 打刻画面（ホーム）

| 操作 / UI要素 | 遷移先 | 条件 |
|---|---|---|
| ヘッダー [履歴] | **W-04** 勤務履歴画面 | — |
| ヘッダー [設定] | **W-05** 設定画面 | — |
| [ログアウト] | **S-01** ログイン画面 | 確認ダイアログ → OK |
| [出勤] ボタン | 遷移なし（画面内でステータス更新） | 未出勤の場合のみ有効 |
| [退勤] ボタン（未提出なし） | 遷移なし（画面内でステータス更新） | 勤務中かつ未提出報告なし |
| [退勤] ボタン（未提出あり） | 確認ダイアログ「未提出の作業報告があります。退勤しますか？」 | 勤務中かつ未提出報告あり |
| 退勤確認 → [報告してから退勤] | **W-03** 作業報告画面 | — |
| 退勤確認 → [このまま退勤] | 遷移なし（退勤処理を実行） | — |
| [現場開始] ボタン | **W-02** 現場選択画面 | 勤務中かつ現場作業中でない場合 |
| [現場終了] ボタン | 遷移なし（画面内でステータス更新）→ 作業報告の促し表示 | 現場作業中の場合のみ有効 |
| 作業報告の促し → [報告する] | **W-03** 作業報告画面 | 現場終了後に表示される |
| 作業報告の促し → [後で] | 遷移なし（促しを閉じる。未提出バッジを表示） | — |
| ⚠️ 未提出バッジ → [報告する] | **W-03** 作業報告画面 | 未提出の報告がある場合のみ表示 |
