# SPEC-005: 勤務実績画面

## 概要
管理者がA-02画面で月単位の勤務実績を確認し、
給与計算用・人事評価用のCSVをエクスポートする。

## 対応画面
- A-02: 勤務実績画面

## 対応ドキュメント
- `docs/design/csv-format.md` — CSVエクスポート仕様

---

## 受入基準

### 勤務実績一覧
- [ ] 月単位で全作業員の勤務実績が一覧表示される
- [ ] [◀] / [▶] で月を切り替えられる
- [ ] 各作業員の行に、月間の勤務日数・総勤務時間・確定勤務時間が表示される
- [ ] 作業員の行をタップすると A-03（作業員詳細画面）に遷移する
- [ ] [← 戻る] で A-01（管理ダッシュボード）に戻れる

### CSVエクスポート
- [ ] [CSVエクスポート] ボタンで表示中の月のCSVをダウンロードできる
- [ ] CSV①（給与計算用）とCSV②（人事評価用）の2種類を出力できる
- [ ] ファイル名は `勤務実績_YYYY年MM月.csv` 形式
- [ ] 文字コードは UTF-8（BOM付き）※ Excelで文字化けしないため
- [ ] 改行コードは CRLF
- [ ] 並び順は 作業員名 → 日付 の昇順

## データモデル

### 一覧表示用データ

```javascript
MonthlyWorkerSummary {
  workerId: string,
  name: string,
  workDays: number,           // 勤務日数
  totalCalculatedHours: number,  // 打刻ベースの合計勤務時間
  totalConfirmedHours: number,   // 確定勤務時間の合計
  overtimeHours: number          // 残業時間の合計（30分単位）
}
```

### CSV出力
`docs/design/csv-format.md` に定義済みのカラム定義に従う。

#### CSV①: 給与計算用
| カラム | 内容 |
|--------|------|
| 作業員名 | Worker.name |
| 作業員ID | Worker.userId |
| 日付 | YYYY-MM-DD |
| 曜日 | 月〜日 |
| 出勤時刻 | HH:MM |
| 退勤時刻 | HH:MM |
| 総勤務h | 退勤 - 出勤（小数） |
| 休憩h | 打刻なし。管理者が確定勤務時間で調整済み |
| 実労働h | 打刻ベースの労働時間 |
| 確定勤務h | confirmedHours |
| 現場数 | SiteStartの回数 |
| 移動h | 現場間の空き時間合計 |

> 休憩の打刻は不要と決定（2026-07-21）。
> 休憩hカラムは空欄またはハイフンで出力。
> 給与計算には「確定勤務h」を使用する。

#### CSV②: 人事評価用
| カラム | 内容 |
|--------|------|
| 作業員名 | Worker.name |
| 作業員ID | Worker.userId |
| 日付 | YYYY-MM-DD |
| 現場名 | Property.name |
| 清掃難易度 | Property.difficulty |
| 作業h | SiteEnd - SiteStart |
| 担当箇所 | WorkReport.areas（セミコロン区切り） |
| 評価ランク | WorkReport.evaluationRank |
| 評価備考 | WorkReport.evaluationNote |
| 備考（翻訳） | WorkReport.notes.translated |
| 備考（原文） | WorkReport.notes.original |

## UI仕様（A-02）

### レイアウト構成
- **ヘッダー**: [← 戻る] + 「勤務実績」タイトル
- **月ナビゲーション**: [◀ 前月] 2026年7月 [翌月 ▶]
- **サマリーエリア**: 月間の合計勤務日数・合計勤務時間
- **実績テーブル**:

| 作業員名 | 勤務日数 | 計算時間 | 確定時間 | 残業 |
|---------|---------|---------|---------|------|
| 田中太郎 | 20日 | 162.5h | 160.0h | 8.0h |
| マリク | 22日 | 175.0h | 176.0h | 16.0h |

- **エクスポートボタン**: [CSVエクスポート]

## エラーハンドリング

| 条件 | 動作 | メッセージ |
|------|------|-----------|
| 該当月にデータなし | 空の一覧を表示 | 「この月の勤務実績はありません」 |
| CSV出力時にデータなし | ボタン非活性 or ダイアログ | 「出力するデータがありません」 |

---

## 休憩時間に関する方針（2026-07-21 確定）

- 休憩時間の打刻は **不要**
- CSV①の「休憩h」カラムは空欄（`-`）で出力
- 実際の勤務時間の調整は、管理者が「確定勤務時間」で行う
- 例: 打刻上8.5時間 → 休憩1時間を差し引いて7.5時間を確定
