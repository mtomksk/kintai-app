# SPEC-004: 管理ダッシュボード

## 概要
管理者がA-01画面で全作業員の当日ステータスを一覧確認し、
各作業員の勤務時間を確定する（給与計算に直結）。
また、管理者自身も「自分の打刻」タブから出退勤を記録できる。

## 対応画面
- A-01: 管理ダッシュボード

## 対応ユースケース・ルール
- UC-05: 管理者が勤務実績を確認・確定する
- UC-10: 管理者が仮登録を紐付ける（概要のみ。詳細はPhase 2以降）
- 境界コンテキスト「勤怠管理」ルール#9, #12, #13

---

## 受入基準

### 管理タブ
- [ ] 全作業員の当日の勤務ステータスが一覧表示される
- [ ] 各作業員の出勤時刻、退勤時刻、現在のステータスが確認できる
- [ ] 作業員をタップすると、A-03（作業員詳細画面）に遷移する
- [ ] 作業員ごとに勤務時間を入力/選択して [確定] できる
- [ ] 確定した勤務時間が DailyRecord.confirmedHours に保存される
- [ ] 残業は30分単位で管理される
- [ ] 「案件情報未確定」の仮登録がある場合、一覧上部に通知として表示される

### 自分の打刻タブ
- [ ] 管理者自身の打刻画面（W-01と同等の機能）が表示される
- [ ] 出勤/退勤/現場開始/現場終了がSPEC-002, SPEC-003と同様に動作する

### ナビゲーション
- [ ] フッターメニューから A-02（勤務実績）、A-04（現場管理）、A-05（割り振り）、A-06（アカウント管理）に遷移できる
- [ ] ログアウトボタンでS-01に戻れる

## 操作フロー

### 勤務時間の確定（正常系）

| # | ユーザー操作 | システム動作 |
|---|------------|------------|
| 1 | 管理ダッシュボードを開く | — |
| 2 | — | 全作業員の当日ステータス一覧を表示 |
| 3 | 作業員の勤務時間入力欄に値を入力 | — |
| 4 | [確定] をタップ | — |
| 5 | — | confirmedHours を DailyRecord に保存 |
| 6 | — | 「確定しました」トースト表示 |

### 勤務時間の入力方式

```
  方式: 数値入力 + 30分単位のバリデーション

  例:
  - 打刻上の勤務時間: 8時間23分（出勤09:00 → 退勤17:23）
  - 管理者の確定入力: 8.0 時間（30分単位に丸め）
  - 確定ボタンで保存

  ※ 入力値は 0.5 単位で入力可能（0.5, 1.0, 1.5, ... 12.0 等）
```

---

## エラーハンドリング

| 条件 | 動作 | メッセージ |
|------|------|-----------|
| 確定時間が0.5単位でない | 入力エラー | 「勤務時間は30分単位で入力してください」 |
| 確定時間が24時間超 | 入力エラー | 「勤務時間を確認してください」 |
| 確定後の再変更 | 上書き保存 | 「確定済みの勤務時間を変更します。よろしいですか？」 |

## データモデル

SPEC-002の `DailyRecord` に加え、確定勤務時間を使用：

```javascript
DailyRecord {
  workerId: string,
  date: string,
  status: WorkStatus,
  timeStamps: TimeStamp[],
  confirmedHours: number | null   // 管理者が確定した勤務時間（0.5単位, 例: 8.0）
}
```

### ダッシュボード表示用データ

```javascript
WorkerDailyStatus {
  workerId: string,
  name: string,
  status: WorkStatus,
  clockIn: string | null,         // 出勤時刻
  clockOut: string | null,        // 退勤時刻
  currentJobName: string | null,  // 現場作業中の場合、案件名
  totalCalculated: number,        // 打刻ベースの計算勤務時間（分）
  confirmedHours: number | null   // 管理者確定済みの勤務時間
}
```

## UI仕様（A-01）

### レイアウト構成
- **ヘッダー**: アプリ名、[設定] アイコン
- **タブ切替**: [管理] / [自分の打刻]
- **管理タブ**:
  - 日付表示（当日）
  - 「案件情報未確定」通知エリア（仮登録がある場合のみ）
  - 作業員ステータス一覧テーブル:
    | 名前 | ステータス | 出勤 | 退勤 | 計算時間 | 確定時間 | 操作 |
    |------|----------|------|------|---------|---------|------|
  - 各行に確定時間の入力欄 + [確定] ボタン
- **自分の打刻タブ**: W-01（打刻画面）と同じUI
- **フッターメニュー**: [勤務実績] [現場管理] [割り振り] [アカウント]
