# ドメインモデル図

> 業務に登場する「もの」と「その関係」を表す図。
> データベース設計図（ER図）ではなく、業務の構造を表す。

---

## 全体モデル図

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
```

## エンティティと値オブジェクトの分類

| 概念 | 種類 | 説明 |
|------|------|------|
| Account | エンティティ | ログイン情報。ユーザーを一意に識別 |
| Worker | エンティティ | 作業員の情報 |
| Site | エンティティ | 現場（物件）の情報 |
| DailyRecord | 集約ルート | 1日分の打刻記録のまとまり |
| TimeStamp | エンティティ | 個々の打刻記録 |
| WorkOrder | エンティティ | 作業依頼 |
| WorkReport | エンティティ | 作業報告 |
| Location | 値オブジェクト | GPS座標（lat, lng） |
| WorkStatus | 値オブジェクト | 勤務ステータス（未出勤/勤務中/現場作業中/退勤済み） |
| CleaningArea | 値オブジェクト | 担当箇所（台所、浴室等） |
| TranslatedText | 値オブジェクト | 翻訳付きテキスト（original, originalLang, translated, translatedLang） |
