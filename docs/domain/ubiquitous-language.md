# ユビキタス言語集（Ubiquitous Language Glossary）

> 業務の専門家と開発者が「同じ言葉」で会話するための辞書。
> コード内の変数名・クラス名もこの言葉をそのまま使う。
> 新しい用語が出てきたら必ずここに追加する。

---

## 人に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 作業員 | Worker | 清掃作業を行う人員 | 正社員・パート問わず |
| 管理者 | Manager | 作業員の勤怠管理・現場割り振りを行う人 | なおえさん、ひろひささん |
| リーダー | Leader | 外国人作業員への通訳・伝達を担う人 | マリクさん（シンハラ語対応） |
| ユーザー | User | システムを使う人の総称（作業員＋管理者） | — |
| ロール | Role | ユーザーの権限種別 | worker / admin |

## 勤怠に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 打刻 | TimeStamp | 時刻を記録する行為、またはその記録 | 出勤・退勤・現場開始・現場終了の4種類 |
| 出勤 | ClockIn | 業務開始の打刻 | 1日に1回（通常） |
| 退勤 | ClockOut | 業務終了の打刻 | 1日に1回（通常） |
| 現場開始 | SiteStart | ある現場での作業開始の打刻 | 1日に複数回あり得る |
| 現場終了 | SiteEnd | ある現場での作業終了の打刻 | — |
| 勤務ステータス | WorkStatus | 作業員の現在の状態 | 未出勤 / 勤務中 / 現場作業中 / 退勤済み |
| 日次記録 | DailyRecord | ある作業員の1日分の打刻をまとめたもの | — |
| 勤務時間 | WorkingHours | 出勤〜退勤の間の合計時間 | 休憩を除く |
| 移動時間 | TravelTime | 現場間の移動にかかった時間 | 現場終了〜次の現場開始の間 |

## 現場に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 現場 | Site | 清掃作業を行う物件・場所 | 住所、依頼元（不動産会社）を持つ |
| 作業依頼 | WorkOrder | 不動産会社からの清掃依頼 | FAXで受領（将来的に電子化） |
| 作業報告 | WorkReport | 現場ごとの作業完了報告 | 写真（最大10枚）、備考を含む |
| 担当箇所 | CleaningArea | 現場内の清掃担当エリア | 台所、浴室、リビング等 |
| 清掃難易度 | CleaningDifficulty | 現場の清掃の難しさ | 人事考課にも影響 |

## 報告・評価に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 人事考課 | PerformanceReview | 作業員の評価 | スピード、品質、担当箇所で評価 |
| 清掃品質 | CleaningQuality | 清掃の仕上がり評価 | 清掃難易度に対する仕上がり |
| 清掃スピード | CleaningSpeed | 作業にかかった時間 | 作業時間で計測 |

## システムに関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 位置情報 | Location | GPS座標（緯度・経度） | 打刻時に記録 |
| 共有端末 | SharedDevice | 複数の作業員が共用するスマートフォン | アカウント切替で使用 |
| アカウント | Account | システムへのログイン情報 | 端末ではなく個人に紐づく |

## 多言語・翻訳に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 翻訳付きテキスト | TranslatedText | 原文と翻訳文をセットで保持するデータ構造 | 備考欄等の自由入力に使用 |
| 国際化 | i18n | UIのラベルやメッセージを複数言語で表示する仕組み | 翻訳ファイル（JSON）で管理 |
| 表示言語 | DisplayLanguage | ユーザーごとのUI表示言語設定 | ja / si / en |
| 翻訳API | TranslationAPI | 自由入力テキストを別言語に自動変換するサービス | Google Cloud Translation を使用 |
