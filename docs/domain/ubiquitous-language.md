# ユビキタス言語集（Ubiquitous Language Glossary）

> 業務の専門家と開発者が「同じ言葉」で会話するための辞書。
> コード内の変数名・クラス名もこの言葉をそのまま使う。
> 新しい用語が出てきたら必ずここに追加する。

---

## 人に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 作業員 | Worker | 清掃作業を行う人員 | 正社員・パート問わず |
| 管理者 | Manager | 作業員の勤怠管理・現場割り振りを行う人 | なおえさん、ひろひささん |
| リーダー | Leader | 外国人作業員への通訳・伝達を担う人 | マリクさん（シンハラ語対応） |
| ユーザー | User | システムを使う人の総称（作業員＋管理者） | — |
| ロール | Role | ユーザーの権限種別 | worker / admin |
| チーム | Team | 現場作業を行う作業員のグループ | 2〜4チームが並行稼働することがある |

## 勤怠に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 打刻 | TimeStamp | 時刻を記録する行為、またはその記録 | 出勤・退勤・現場開始・現場終了の4種類 |
| 出勤 | ClockIn | 業務開始の打刻 | 1日に1回（通常） |
| 退勤 | ClockOut | 業務終了の打刻 | 1日に1回（通常） |
| 現場開始 | SiteStart | ある現場での作業開始の打刻 | 1日に複数回あり得る |
| 現場終了 | SiteEnd | ある現場での作業終了の打刻 | — |
| 勤務ステータス | WorkStatus | 作業員の現在の状態 | 未出勤 / 勤務中 / 現場作業中 / 退勤済み |
| 日次記録 | DailyRecord | ある作業員の1日分の打刻をまとめたもの | — |
| 勤務時間 | WorkingHours | 出勤〜退勤の間の合計時間 | 休憩を除く |
| 確定勤務時間 | ConfirmedHours | 管理者が確認・確定した実際の勤務時間 | 打刻の間違いやズレを補正。給与計算に直結 |
| 残業 | Overtime | 所定労働時間を超えた勤務 | 30分単位で管理 |
| 移動時間 | TravelTime | 現場間の移動にかかった時間 | 現場終了〜次の現場開始の間 |
| 給与締め日 | PayrollClosingDate | 給与計算の締め日 | 毎月10日 |

## 現場に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 物件 | Property | 清掃作業を行う建物・場所の継続的な情報 | 住所、棟名、部屋番号、リピートフラグを持つ。繰り返し利用される |
| 案件 | Job | 物件に対する個別の清掃作業の単位 | 作業予定日、担当者、鍵情報、作業指示書を持つ。作業日ごとに発生 |
| 案件ID | JobId | 案件を一意に識別するID | 日付だけでは一意にならないため、個別IDが必要 |
| 案件ステータス | JobStatus | 案件の進行状態 | 未着手 / 作業中 / 完了 / 案件情報未確定 |
| 現場（旧称） | Site | 物件（Property）の別名。旧呼称 | コード上はPropertyを使用 |
| 作業依頼 | WorkOrder | 不動産会社からの清掃依頼 | FAXで受領。画像添付で電子化 |
| 作業報告 | WorkReport | 案件ごとの作業完了報告 | 写真（証拠写真含む）、備考を含む |
| 担当箇所 | CleaningArea | 現場内の清掃担当エリア | 台所、浴室、リビング等 |
| 清掃難易度 | CleaningDifficulty | 現場の清掃の難しさ | 広さだけでなく汚れ具合にも左右。事前判断は困難 |
| 依頼元 | Client | 清掃を依頼する不動産会社等 | — |

## 鍵に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 鍵情報 | KeyInfo | 現場の鍵の受け取り方法に関する情報 | 受け取り方法＋備考のセット |
| 鍵受け取り方法 | KeyPickupMethod | 鍵の受け取り手段 | 現場設置 / 管理者から当日受領 / 顧客から直接受領 / 作業指示書確認 / その他 |
| 鍵備考 | KeyNote | 鍵に関する自由入力のメモ | 例：「建物裏のキーボックス」「ダイヤル番号：○○○○」 |

## 作業指示書に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 作業指示書 | InstructionSheet | FAXで受領する清掃依頼の詳細資料 | 棟情報、鍵情報、清掃依頼内容を含む。日本語のまま作業員へ共有 |
| 指示書画像 | InstructionImage | 作業指示書を撮影/スキャンした画像ファイル | JPG/PNG/PDF対応。案件完了後に一定期間で削除可能 |

## 物件管理に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| リピートフラグ | RepeatFlag | 過去に複数回作業した物件、または今後も依頼が見込まれる物件への印 | 管理者が付与。一覧での絞り込みに使用 |
| 物件データ | PropertyData | 継続保存される物件の基本情報 | 住所、棟名、部屋番号、Googleマップ用住所、リピートフラグ |
| 案件データ | JobData | 作業日ごとに発生する情報 | 作業予定日、担当者、鍵情報、作業指示書画像、ステータス |

## 報告・評価に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 人事考課 | PerformanceReview | 作業員の評価 | 安全・速さ・品質・対応可能な作業の幅を重視 |
| 作業評価ランク | EvaluationRank | 管理者が物件単位の作業結果に付けるランク | 管理者が作業後の現場確認時に実施 |
| 評価備考 | EvaluationNote | ランク付けの理由を記録する自由入力欄 | 「なぜBなのか」等の判断根拠 |
| 清掃品質 | CleaningQuality | 清掃の仕上がり評価 | 清掃難易度に対する仕上がり |
| 清掃スピード | CleaningSpeed | 作業にかかった時間 | 作業時間で計測 |
| 証拠写真 | EvidencePhoto | 作業前の破損等を記録する写真 | 完了報告用写真は不要だが、エビデンスとしての写真は添付 |
| 給与明細 | PaySlip | 月次の給与明細書 | ExcelでPDF化したもの。アプリでは閲覧機能のみ提供 |

## システムに関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 位置情報 | Location | GPS座標（緯度・経度） | 打刻時に記録 |
| 共有端末 | SharedDevice | 複数の作業員が共用するスマートフォン | アカウント切替で使用 |
| アカウント | Account | システムへのログイン情報 | 端末ではなく個人に紐づく |
| 仮登録 | ProvisionalRegistration | 未登録の現場を作業員が簡易入力して暫定的に記録すること | 管理者が後から正式な物件/案件と紐付ける |
| ページネーション | Pagination | 大量データを一定件数ごとにページ分割して表示する方式 | 物件一覧等で使用。1ページ20〜50件目安 |

## 多言語・翻訳に関する用語

| 用語（日本語） | 英語 / コード名 | 定義 | 補足 |
|---------------|----------------|------|------|
| 翻訳付きテキスト | TranslatedText | 原文と翻訳文をセットで保持するデータ構造 | 備考欄等の自由入力に使用 |
| 国際化 | i18n | UIのラベルやメッセージを複数言語で表示する仕組み | 翻訳ファイル（JSON）で管理 |
| 表示言語 | DisplayLanguage | ユーザーごとのUI表示言語設定 | ja / si / en |
| 翻訳API | TranslationAPI | 自由入力テキストを別言語に自動変換するサービス | Google Cloud Translation を使用 |
