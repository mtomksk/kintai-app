## 1日の業務フロー（全体像）

```
  ┌─────────────────────────────────────────────────────────────────┐
  │                                                                 │
  │   ① 朝ミーティング（事務所）                                     │
  │   ┌───────────────────────────────────────────────┐             │
  │   │ ・当日の現場割り振りの確認                       │             │
  │   │ ・❓ 鍵の受け取り                               │             │
  │   │ ・❓ 作業指示書（担当箇所）の配布                 │             │
  │   │ ・❓ その他の共有事項                            │             │
  │   └───────────────────────────────────────────────┘             │
  │          │                                                      │
  │          ↓                                                      │
  │   ② 現場へ移動                                                  │
  │   ┌───────────────────────────────────────────────┐             │
  │   │ ・❓ 移動手段（電車？車？）                      │             │
  │   │ ・❓ 最寄り駅からマップアプリで現場へ              │             │
  │   │ ・❓ 交通費は自己負担？会社負担？                 │             │
  │   └───────────────────────────────────────────────┘             │
  │          │                                                      │
  │          ↓                                                      │
  │   ③ 現場作業                                                    │
  │   ┌───────────────────────────────────────────────┐             │
  │   │ ・❓ 現場到着時に管理者へ連絡（電話？）           │             │
  │   │ ・鍵で入室し、作業開始                           │             │
  │   │ ・指示書に基づいて清掃作業                       │             │
  │   │ ・作業完了後、写真撮影                           │             │
  │   │ ・❓ 管理者へ完了の連絡（電話？）                 │             │
  │   │ ・❓ 管理者の承認を待つ？                        │             │
  │   └───────────────────────────────────────────────┘             │
  │          │                                                      │
  │          ├──→ 次の現場がある場合 → ② に戻る                     │
  │          │                                                      │
  │          ↓                                                      │
  │   ④ 退勤                                                       │
  │   ┌───────────────────────────────────────────────┐             │
  │   │ ・❓ 事務所に戻る？直帰？                        │             │
  │   │ ・❓ 退勤の連絡方法（電話？）                     │             │
  │   └───────────────────────────────────────────────┘             │
  │                                                                 │
  └─────────────────────────────────────────────────────────────────┘
```

## 詳細フロー：管理者 vs 作業員

```
  時間軸    管理者                          作業員
  ──────   ───────────────────             ───────────────────

  朝       ┌─────────────────┐             ┌─────────────────┐
           │ 朝ミで指示       │←── 対面 ──→│ 朝ミに参加       │
           │ ・現場割り振り    │             │ ・現場確認       │
           │ ・鍵の受け渡し❓  │             │ ・鍵受取❓       │
           │ ・指示書配布❓    │             │ ・指示書受取❓    │

                    │                               │

           │ ・連絡対応        │             │                 │
```
