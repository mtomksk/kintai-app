# 勤怠管理アプリ（kintai-app）

清掃会社向けの勤怠管理Webアプリケーションです。

## 概要

- 作業員の出勤・退勤・現場作業の打刻管理
- 管理者による勤務実績の確認・集計
- 多言語対応（日本語・シンハラ語・英語）

## 技術スタック

- フロントエンド: HTML + CSS + JavaScript
- データ保存: （検討中）
- バージョン管理: Git + GitHub
